dual-cellex — Quiz 輸入功能

左欄輸入框 × 右欄遮罩答案，按 ENTER 揭開並比對

範例一：輸入在左，答案在右（預設）
左欄為輸入框，右欄為遮罩答案欄。按 ENTER 後遮罩揭開，輸入框鎖定。
答對顯示 綠色，答錯顯示 紅色 + 震動動畫。
大小寫敏感比對。
Acetylsalicylic Acid Amoxicillin Metformin Atorvastatin
HTML 寫法：<dual-col input-quiz quiz-placeholder="提示"></dual-col>
答案欄：<dual-col answer-quiz overlay-1-text="？" overlay-1-color="sky">答案</dual-col>
範例二：答案在左，輸入在右（左右對調）
左欄為遮罩答案欄，右欄為輸入框。只需交換欄的順序即可，邏輯完全相同。

範例三：三欄模式（題目 ＋ 輸入 ＋ 答案）
第一欄為靜態題目說明，第二欄為輸入框，第三欄為遮罩答案。
col-widths="2:2:1" 控制欄寬比例。
阿斯匹靈的藥物分類（英文） NSAID 二甲雙胍的最大單次劑量（mg） 1000 Penicillin 中文名稱 盤尼西林 阿斯匹靈學名縮寫 ASA
三欄模式：第一欄靜態題目（show-menu="false"）， 第二欄 input-quiz，第三欄 answer-quiz。
PHP Class 寫法（DualCell::rowQuiz()）
以下為等效的 PHP Class 寫法，適合從資料庫動態產生測驗題： 
// 輸入在左（預設）
DualCell::rowQuiz('輸入藥物學名…', 'Metformin', [
    'overlay_text'  => '？',
    'overlay_color' => 'safe',
]);

// 輸入在右
DualCell::rowQuiz('輸入答案…', 'Aspirin', [
    'input_side'   => 'right',
    'overlay_text' => '？',
    'col_widths'   => '1:1',
]);

// 從資料庫批次產生
$quiz_items = $pdo->query("SELECT prompt, answer FROM quiz_drug")->fetchAll();
foreach ($quiz_items as $q):
    echo DualCell::rowQuiz($q['prompt'], $q['answer'], ['overlay_color' => 'safe']);
endforeach;
